## Install

Install Python Liquid Babel with [pip](https://pip.pypa.io/en/stable/getting-started/):

```plain
$ python -m pip install -U liquid-babel
```

Or [pipenv](https://pipenv.pypa.io/en/latest/):

```plain
$ pipenv install liquid-babel
```
